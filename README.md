Stachu OS 4.0.0
Based on Stachu OS Zima kernel

Made with
- tkinter
- customtkinter
- ctk components
- ctk color picker
- customtkinter builder

Portable or Normal installation
Normal

The Normal installation has everything in multiple python files (.py)
The file thats meant to be started first is root.py to boot StachuOS.
This installation is recommended for StachuOS app developers or modders

Portable

The portable installation contains everything in one executable (.exe) file
might run slightly faster compared to the Normal installation.
Modding is not available in this installation
Recommended for casual users
